Javascript Frameworks

Angular JS

Informatii Generale

Probabil primul framework care a atins un grad de acoperire si utilizare mai mare decat jQuery. Pornit undeva in Google, se afla pe ultimele picioare, desi se mai gasesc aplicatii multe care folosesc Angular JS, devine problematica mentinerea iar suportul oficial al echipei de dezvoltare se va termina in curand.

Printre avantajele folosirii Angular JS se numara:

  • Apropiat de JavaScript, nu necesita Typescript sau Babel
  • Framework complet, contine orice de la comunicarea cu serverul la stocarea cookies
  • Functionalitati puternice de validat formulare.

Dezvantajele folosirii Angular JS cuprind:

  • Concepte invechite greu de scalat si intretinut
  • Suportul urmeaza sa se termine
  • Se gasesc greu dezvoltatori dispusi sa migreze sau sa mai creeze functionalitati noi

CSS

La un moment dat vom intra si in CSS, unde vom invata sa coloram fundaluri, texte, sa lucram cu dimensiunile textelor, folosind proprietati precum:

  • color
  • background-color
  • font-size
  • text-decoration
  • Font-weight

Apoi vom trece la concepte putin mai avansate cum ar fi:

  1. box model
  2. stacking context
  3. pozitionari
  4. float
  5. layout avansat

JavaScript

In viitor ne putem uita si la JavaScript unde vom invata detaliile programarii in general cu notiuni de programare si stiinta calculatoarelor, cum ar fi:

  • variabile
  • bucle
  • blocuri de decizie
  • functii
  • obiecte

JavaScriptul este in esenta un limbaj de scripting pentru browsere in principal, prin urmare cu el se manipuleaza site-urile web si se creeaza aplicatiile web, folosind tehnici precum:

  1. Ajax
  2. DOM Scripting
  3. Validarea de formulare
  4. Afisarea de dialoguri
  5. Procesarea datelor

Node JS

Ultima provocare pentru un dezvoltator incepator este node js. Node js este JavaScript pe server. Avantajele folosirii node sunt:

  1. Acelasi limbaj de programare si pe client si pe server, cod re-utilizabil la maxim.
  2. Viteza buna (intre Go si PHP)
  3. Sistemul asincron puternic din JavaScript
  4. Comunicare simpla cu clientii prin websockets, Ajax

Dezvantajele folosirii node sunt:

  • Usor de invatat, greu de stapanit
  • Sistemul asincron este greu de stapanit
  • Numerele in JavaScript se reprezinta greu, facand aplicatiile ce folosesc bani greu de realizat.
